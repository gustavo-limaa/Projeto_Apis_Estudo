# 🚀 Backend Lab: Evolução & Arquitetura .NET

Este repositório centraliza meus projetos de estudo em C#, servindo como um laboratório prático para aplicação de **Clean Architecture**, **SOLID** e **Testes Automatizados**.

## 📂 Estrutura do Repositório

O repositório está dividido em projetos que demonstram diferentes estágios de maturidade técnica:

### 🛠️ Em Desenvolvimento Ativo
* **[Prateleira_Universal](./Prateleira_Universal):** Sistema principal de gerenciamento de mídias. 
    * **Foco atual:** Integração com APIs via **Refit**, **Testes de Integração** e implementação de **Design Patterns**.

### ♻️ Projetos em Fila para Refatoração (Clean Code & Data Arch)
Estes projetos foram desenvolvidos inicialmente para funcionalidade e agora estão sendo migrados para padrões de arquitetura limpa:
* **[Cadastro](./Cadastro):** Módulo de gestão de registros.
* **[FilmeApis](./FilmeApis):** Consumo de catálogos cinematográficos.
* **[API de usuários](./API%20de%20usuários):** Autenticação e perfis.
* **[Armazenamento (Em breve)](#):** Persistência e manipulação de arquivos.

---

## 🧪 Pilares do Aprendizado

1. **Testes:** Implementação de testes unitários e de integração com **xUnit**, **Bogus** e **Moq**.
2. **Arquitetura:** Transição de modelos anêmicos para **Domain-Driven Design (DDD)** e **Clean Architecture**.
3. **Consumo de Dados:** Uso de **Refit** para comunicações resilientes e tipadas entre serviços.
4. **Segurança:** Gestão de chaves sensíveis através de **User Secrets** e variáveis de ambiente.

## 🛠️ Tecnologias Principais
`.NET 8/10` | `C#` | `Entity Framework` | `Refit` | `Scalar/Swagger`
